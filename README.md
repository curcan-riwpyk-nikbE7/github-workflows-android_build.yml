# DriveOS Car Launcher (Full Project)

Этот архив содержит полную структуру проекта для сборки APK через Capacitor.

## Как собрать:
1. Распакуйте архив.
2. Загрузите файлы в свой репозиторий GitHub.
3. Используйте GitHub Actions (файл workflow из предыдущего сообщения) для автоматической сборки APK.

## Стек:
- React + Vite
- Tailwind CSS
- Capacitor (для Android)
- Zustand
